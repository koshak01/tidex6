# Week 2 — Weekly Video Update (1 минута)

**Нарратив:** На прошлой неделе показали deposit/withdraw через CLI. Теперь — то же самое через сайт tidex6.com.

---

## Подготовка (до записи)

1. Открой **tidex6.com**, кошелёк **уже подключён**, инвайт **уже одобрен** — экран с Deposit + Withdraw
2. Скопируй в заметки:
   ```
   auditor_public_key:  061ec073d77cf0c7326c0c7974b426ac9e71e8529c6506af5678b937930e7a27
   memo:                Rent April 2026
   ```
3. Убедись: баланс >= 0.2 SOL
4. Закрой лишние вкладки и уведомления

---

## Сценарий (60 секунд)

### Часть 1: Deposit (0:00 — 0:30)

| Сек | Действие |
|-----|----------|
| 0:00 | Экран с Deposit + Withdraw панелями. Зритель видит интерфейс |
| 0:03 | Нажми **"0.1 SOL"** |
| 0:05 | Кликни в **Memo**, вставь `Rent April 2026` (Cmd+V) |
| 0:08 | Кликни в **Auditor public key**, вставь hex ключ (Cmd+V) |
| 0:12 | Нажми **"Deposit"** |
| 0:14 | Phantom попап — нажми **подтвердить** |
| 0:17 | Ожидание подтверждения (Activity Log обновляется) |
| 0:25 | Note появляется — **скопируй её** (Cmd+C) |
| 0:28 | Пауза 2 сек — зритель видит успех |

### Часть 2: Withdraw (0:30 — 1:00)

| Сек | Действие |
|-----|----------|
| 0:30 | Кликни в поле **Note** (панель Withdraw) |
| 0:32 | Вставь note (Cmd+V) |
| 0:34 | Recipient уже заполнен — не трогай |
| 0:35 | Нажми **"Withdraw"** |
| 0:37 | Proof generation начинается (Activity Log показывает прогресс) |
| 0:45 | Phantom попап — нажми **подтвердить** |
| 0:48 | Ожидание подтверждения |
| 0:55 | Успех в Activity Log — withdraw завершён |
| 0:58 | Пауза 2 сек — финал |

## Тайминг-риски

| Этап | Реальное время | Проблема |
|------|---------------|----------|
| Deposit confirm | 5-15 сек | Если дольше 10 сек — вырежи ожидание в редакторе |
| Proof generation | 15-30 сек | **Главный риск.** Если дольше 15 сек — ускорь 2x в редакторе |

**Совет:** запиши в реальном времени (~2-3 мин), потом ускорь ожидания в редакторе до 1 минуты.

## Советы

1. **Не печатай вручную** — всё через Cmd+V из заметок
2. **Курсор двигай плавно** — не метайся
3. **Ожидание proof** — можно ускорить при монтаже
4. **Платформа:** YouTube, Loom или Vimeo
