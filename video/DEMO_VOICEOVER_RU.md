# tidex6 — Текст для озвучки Demo Video (русский)

**Общее время:** ~90 секунд
**Формат:** читаешь в диктофон с секундомером, потом ElevenLabs переведёт
**Темп:** спокойный, не торопись. Лучше 100 секунд уверенно чем 80 скороговоркой.

---

## Как работать с этим файлом

1. Открой секундомер на телефоне
2. Открой диктофон (Voice Memos на iPhone или QuickTime Audio на Mac)
3. Запусти оба одновременно
4. Читай текст, сверяясь с таймингом
5. Если сбился — не останавливайся, дочитай до конца. Можно вырезать потом.
6. Запиши 3 дубля, выбери лучший

## 📋 Чеклист перед записью

### Подготовка
- [ ] Секундомер на телефоне открыт
- [ ] Диктофон открыт (Voice Memos или QuickTime Audio)
- [ ] Текст распечатан или на втором экране (не читай с того же экрана что запись)
- [ ] В комнате тихо — нет гула кондиционера, холодильника, улицы

### Голос
- [ ] Говори **спокойно**, как объясняешь другу за кофе
- [ ] Не читай — **рассказывай**. Допускаются мелкие отклонения от текста
- [ ] Делай **паузы** где написано — они важны, ElevenLabs их сохранит
- [ ] **Не шепчи** и **не кричи** — нормальный разговорный тон
- [ ] Последнюю фразу «I grant access, not permission» скажи **по-английски** даже в русской записи — ElevenLabs оставит как есть

### После записи
- [ ] Послушай запись — нет ли фоновых шумов
- [ ] Длина ~90 секунд (±15 сек нормально)
- [ ] Если меньше 70 сек — говоришь слишком быстро, перезапиши
- [ ] Если больше 110 сек — можно, но проверь что не тянешь паузы

## 🔧 Техническое

### Где записывать голос
- **iPhone Voice Memos** — самый простой, AirPods Pro как микрофон
- **QuickTime** → File → New Audio Recording → выбери AirPods Pro или внешний микро
- **Не** встроенный микрофон MacBook — шипит

### Формат файла
- Voice Memos сохраняет `.m4a` — ElevenLabs принимает
- QuickTime сохраняет `.m4a` — ElevenLabs принимает
- Если нужен `.mp3` — откроешь в QuickTime → File → Export As

### ElevenLabs
- Зайди на https://elevenlabs.io
- Speech to Speech или Dubbing
- Загрузи русский `.m4a`
- Выбери English, мужской голос (попробуй "Adam" или "Antoni")
- Скачай результат `.mp3`

### iMovie
- Create New → Movie
- Drag видео экрана на timeline
- Drag английский `.mp3` под видео
- Подвинь аудио по времени чтобы слова совпадали с действиями на экране
- File → Share → File → 1080p
- Upload на YouTube (unlisted для demo, public для pitch)
